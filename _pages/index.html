ADCHEM 2018

Paving the way for the next industrial revolution through data

Our team from the UBC Data Analytics and Intelligent Systems Lab will conduct a series of presentations focused on machine learning algorithms with an emphasis on process control applications.

ADCHEM is a triennial symposium that brings together researchers and professionals worldwide, to facilitate discussions about the development of control and modelling in chemical, biomedical, and other processing systems. For more information, visit the Official ADCHEM 2018 page.

UBC's ADCHEM Workshop

Big Data Analytics

Big data, data analytics, and machine learning have opportunities for diagnostic, prognostics, and decision making in the process industries. A presentation on Big Data Analytics will be done by Dr. Richard Braatz of the Massachusetts Institute of Technology.

Dimensionality Reduction

Dimensionality reduction is the process of reducing the number of variables in a set of data to isolate a certain variables. There are many applications of this to process control, since it would be useful to find lower-dimensional representations of a dataset without losing much of the data.

Classification and Regression

Classification and regression are two of the primary applications of machine learning. In process control, there are many applications for classification and regression, as they allow data sets to be analyzed effectively.
